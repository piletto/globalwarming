# global_warming

# 🌍 Классификатор природных явлений

**Автор:** Софи (София)

Краткое описание: Простая нейросетевая модель, которая определяет одно из 4 природных явлений по фотографии и выводит его подробное описание.

---

## 📝 Описание проекта

**Цель проекта:** Создать инструмент на основе компьютерного зрения для быстрого распознавания признаков глобального изменения климата по обычной фотографии.

**Область применения:** Проект может быть полезен для образовательных целей, для начинающих специалистов в сфере Data Science, а также как демонстрация возможностей нейросетей в экологическом просвещении.

---

## ⚙️ Функции и Особенности

- Распознавание 4 классов: 🔥 Лесные пожары, 🏜️ Засуха, 🧊 Таяние льда, 🌊 Наводнение
- Автоматическое описание явления после определения класса
- Простота использования — достаточно указать путь к фото и запустить скрипт
- Визуальный вывод в консоль с эмодзи и разделителями

---

## 🛠️ Установка и использование

**Шаг 1:** Установите зависимости в терминале:

pip install tf-keras pillow numpy

Шаг 2: Убедитесь, что в папке проекта лежат файлы:

keras_model.h5 — обученная нейросеть
labels.txt — названия классов
predict.py — основной скрипт

Шаг 3: В файле predict.py замените путь на ваше фото:

image = ImageOpen("путь_к_твоему_фото.jpg")

Шаг 4: Запустите скрипт.
